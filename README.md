# 崩坏3
* 游戏区服：全服
* 游戏版本：不限制
* 设备环境：安卓手机(安卓9以上)
* Root：是

# 特点
* 动态抓取offset，故不限游戏版本、游戏区服
* 完全免费，APP无服务器
* 非常安全，不易被检测封号
* 适配arm64-v8a架构
* 支持游戏分包模式

# 功能
* 满星
* 定怪
* 攻速
* 能量
* 移速
* 超级伤害（该功能风险高）

# 食用
* 手机解锁BL并接管超级权限
* 开启zygisk
* 安装打开URC，跟着toast做就行了
